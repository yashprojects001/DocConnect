## 🌟 Overview

**DocConnect** is a modern healthcare dashboard built to make doctor consultation and appointment booking simple, clean, and efficient.

It helps users:
- search doctors by specialization
- book appointments
- manage reports
- track prescriptions
- check symptoms
- access emergency help
